# Budget App

A local-first budgeting and retirement projection app built with Next.js.

## Development

```powershell
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

Primary app code lives in [`app/page.tsx`](app/page.tsx).

## Checks

```powershell
npm run lint
npm run docs:lint
npm run build
```

Use `npm run check` before a larger handoff.

## Markdown Workflow

Project notes live in [`docs/`](docs/). Start with
[`docs/README.md`](docs/README.md) for the workflow.

Use Markdown for:

- Product and calculation decisions.
- Implementation notes for meaningful changes.
- User-facing change summaries.

Key files:

- [`docs/CHANGELOG.md`](docs/CHANGELOG.md)
- [`docs/decisions/`](docs/decisions/)
- [`docs/templates/`](docs/templates/)
